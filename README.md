# Payment_FIVEM
리얼월드 기반 서버팩을 위한 자동후원시스템

디스코드와 연동하여 유저가 편리하게 서버 아이템을 구매할 수 있도록 제작된 봇 입니다.

config 를 연동해 놓았으니, 직접 설정하셔서 편리하게 사용하실 수 있습니다.
